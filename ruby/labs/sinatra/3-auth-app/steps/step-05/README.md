# Step 5: Login System

This step adds session-based login and logout.

## What's Included
- Login form with email/password
- Session-based authentication
- Logout functionality
- Protected dashboard route
- Flash messages for feedback
- Navigation updates based on auth state

## How to Run
```bash
ruby app.rb
```

## Test Flow
1. Register at http://localhost:4567/register
2. Logout at http://localhost:4567/logout
3. Login at http://localhost:4567/login
4. Access dashboard at http://localhost:4567/dashboard

## Access
Open http://localhost:4567 in your browser.
