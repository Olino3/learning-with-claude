# Step 3: User Model

This step adds the User model with basic validations.

## What's Included
- User model with Sequel
- Email and name validations
- Email normalization (lowercase, trim)
- List users endpoint for testing

## How to Run
```bash
ruby app.rb
```

## Test Endpoints
```bash
# List all users
curl http://localhost:4567/users
```

## Access
Open http://localhost:4567 in your browser.
