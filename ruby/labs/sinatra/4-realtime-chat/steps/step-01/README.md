# Step 1: Basic Sinatra Setup

This step creates a basic chat interface with rooms.

## What's Included
- Sinatra web application
- Rooms list page
- Chat room interface (static)
- Basic CSS styling

## How to Run
```bash
ruby app.rb
```

## Test
1. Visit http://localhost:4567/rooms
2. Click on a room to enter
3. The chat interface shows (messages not yet functional)

## Access
Open http://localhost:4567 in your browser.
