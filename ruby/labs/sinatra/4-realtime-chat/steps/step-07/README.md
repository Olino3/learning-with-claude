# Step 7: User Presence (Complete)

This is the complete Real-Time Chat application with all features.

## What's Included
- Online user tracking per room
- User list updates in real-time
- Join/leave notifications
- Message persistence
- Multiple rooms
- Username authentication

## How to Run

**Terminal 1 - WebSocket Server:**
```bash
ruby chat_server.rb
```

**Terminal 2 - Web App:**
```bash
ruby app.rb
```

## Features
- Enter username to join
- See all chat rooms
- Real-time messaging
- Online user list per room
- Join/leave notifications
- Message history

## Test User Presence
1. Open multiple browser windows
2. Enter different usernames
3. Join the same room
4. See the online user list update
5. Close a window - see leave notification

## Access
Open http://localhost:4567 in your browser.
