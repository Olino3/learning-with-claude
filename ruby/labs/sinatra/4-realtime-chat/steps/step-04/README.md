# Step 4: Client-Side WebSocket

This step adds JavaScript WebSocket client.

## What's Included
- WebSocket connection from browser
- Join room on connection
- Send/receive messages
- Auto-reconnect on disconnect

## How to Run

**Terminal 1:**
```bash
ruby chat_server.rb
```

**Terminal 2:**
```bash
ruby app.rb
```

## Test
1. Visit http://localhost:4567/rooms
2. Enter a room
3. Send messages - they appear in real-time!

## Access
Open http://localhost:4567 in your browser.
